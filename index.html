🏡 Smart Home Automation

ESP8266 | Sensors | Blynk App | Real-Time Monitoring

📘 Project Overview

This smart home system automates lights, geysers, and gas valves using NodeMCU ESP8266. Users can control devices via Blynk and monitor temperature/gas levels in real time.

🔧 Components Used

  • ESP8266 (NodeMCU)
  • 3-Channel Relay Module
  • DS18B20 Temperature Sensor
  • MQ2 Gas Sensor
  • Servo Motor (gas valve)
  • Push Buttons
  • Blynk App

⚙️ How It Works

The ESP8266 reads gas and temperature sensors. It sends data to Blynk and reacts based on thresholds. For safety, it shuts off the geyser at 50°C and opens the gas valve using a servo if gas is detected.
